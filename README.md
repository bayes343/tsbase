The public facing readme is located in the `lib/src` directory.  This readme is intended for maintainers / constributors to tsbase.

## Local Development

### Pre-requisites 
- node/npm

### Installation
- run cmd: `npm install`

### Testing
- run cmd: `npm run test` / `karma start`

### Building
- run cmd `npm run build`
